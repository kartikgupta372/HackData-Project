# AuraDesign AI - Premium Design Optimization Toolkit

AuraDesign AI is a state-of-the-art, AI-powered platform designed to revolutionize website conversion rates through intelligent design optimization. Built for performance, aesthetics, and high-conversion results.

## ✨ Core Features

- **Magical UI Components**: Custom-built sparkling text effects and animated interactive elements.
- **Cinematic Preloader**: A branded, smooth reveal mechanism ensuring all heavy assets (3D, large bundles) are ready before view.
- **3D Interactive Scenes**: Integrated Spline 3D scenes with intelligent visibility-based rendering (auto-pause/play when out of view).
- **High-Performance Noise Systems**: CPU-optimized grains and textures using pre-rendered frame cycling.
- **Bento-style Features Grid**: A modern, high-contrast grid system with proximity-based glow effects.
- **Micro-animations**: Momentum-based scrolling and viewport-aware motion sequences using Framer Motion.

## 🚀 Getting Started

Follow these steps to set up the project locally.

### 1. Prerequisites

Ensure you have **Node.js** (v18 or higher) and **npm** installed.

### 2. Fork and Clone

1. **Fork** this repository.
2. Clone your forked repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/AuraDesign-AI.git
   cd AuraDesign-AI
   ```

### 3. Install Dependencies

Install all necessary packages via npm.

```bash
npm install --legacy-peer-deps
```

> [!NOTE]
> We use the `--legacy-peer-deps` flag to ensure compatibility between React 19 and certain high-performance UI components.

### 4. Run the Development Server

Start the local development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

## 🛠️ Integrated Tech Stack
